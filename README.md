# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Запуск

- Скачайте код

- Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

- Добавьте таблицу, созданную по образцу  

- Создайте файл .env и пропишите в нем:

```
WINES_LIST=ваш файл с таблицей
```

- Таблица должна выглядеть как на примере ниже:
<br>
<table>
    <tr>
        <th>Категория</th>
        <th>Название</th>
        <th>Сорт</th>
        <th>Цена</th>
        <th>Картинка</th>
        <th>Акция</th>
    </tr>
    <tr>
        <td>Категроия напитка</td>
        <td>Название напитка</td>
        <td>Сорт напитка</td>
        <td>Цена напитка</td>
        <td>Картинка напитка</td>
        <td>Акция (если есть)</td>
    </tr>
</table>
<br>


- Запустите сайт командой 
```
python main.py runserver
```

- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
